# Улучшение новых изделий

Дата: 2026-07-09

Активная версия:

`versions/KtpnConfigurator_Optimized`

## Что сделано

Запущена работа по промту:

`план развития/CODEX_ПРОМТ_УЛУЧШЕНИЕ_ПРИЛОЖЕНИЯ_2026-07-09.md`

Основной фокус этапа: новые изделия 2КТПН, НКУ, ЩО, ВРУ, КСО и КРУ.

## 1. НКУ, ЩО, ВРУ

Расширена линейка панелей:

- добавлено количество линий в панели;
- добавлен признак учета;
- добавлен признак ОПН;
- типовые составы стали ближе к рабочим:
  - НКУ: вводная, распределительная, управление/собственные нужды;
  - ЩО: две секции, вводы секций, секционный аппарат, линейные панели;
  - ВРУ: ввод, учет, АВР, распределительная панель.

В интерфейсе линейка панелей теперь показывает и редактирует:

- номер;
- секцию;
- тип;
- назначение;
- аппарат;
- бренд;
- модель;
- ток;
- Icu;
- количество линий;
- учет;
- ОПН;
- ширину;
- массу.

## 2. КСО, КРУ

Расширена линейка ячеек:

- добавлен класс точности ТТ;
- добавлена модель ТН;
- добавлен терминал РЗА;
- добавлены отдельные видимые разрывы до и после аппарата;
- добавлен признак заземлителя;
- типовые составы КСО и КРУ стали подробнее.

В интерфейсе линейка ячеек теперь показывает и редактирует:

- номер;
- назначение;
- основной аппарат;
- модель;
- ток;
- ток отключения;
- ТТ;
- класс ТТ;
- ТН;
- модель ТН;
- РЗА;
- терминал;
- разрыв до;
- разрыв после;
- заземлитель;
- ширину;
- массу.

## 3. Расчеты

Уточнены расчеты для новых изделий:

- для НКУ/ЩО/ВРУ расчет габарита идет по сумме ширин панелей;
- добавлена расчетная масса основания для низковольтных изделий;
- добавлен довес на линии, учет и ОПН;
- масса шин НКУ/ЩО/ВРУ теперь считается по току, материалу, длине и количеству секций;
- N/PEN и PE формируются отдельно в результате;
- для КСО/КРУ добавлен довес на выключатели, ТТ, ТН, РЗА и заземлители;
- для КСО/КРУ добавлена расчетная масса шин по длине линейки и номинальному току;
- итоговая ориентировочная масса теперь больше базовой массы и включает довес.

## 4. Проверки

Добавлены и уточнены проверки:

- панель не может быть назначена несуществующей секции;
- ток панели не должен быть выше тока сборных шин;
- количество линий панели не может быть отрицательным;
- для панели учета желательно указать модель узла учета;
- для ячейки с выключателем желательно указать ТТ;
- для вакуумного выключателя проверяются видимые разрывы до и после;
- для ТН проверяется заполнение модели;
- для КРУ сохраняется проверка IAC/LSC/перегородок;
- для КСО сохраняется предупреждение о подтверждении характеристик серией камер.

## 5. Документы

Документы новых изделий стали подробнее:

- приказ и паспорт получают отдельные разделы `Линейка панелей` или `Линейка ячеек`;
- спецификация выводит расширенный состав панелей и ячеек;
- для панелей выводятся секция, тип, назначение, аппарат, ток, Icu, количество линий, учет, ОПН, ширина и масса;
- для ячеек выводятся аппарат, модель, ток, ток отключения, ТТ, класс ТТ, ТН, модель ТН, РЗА, терминал, разрывы, ширина и масса;
- чек-лист для низковольтных изделий дополнен проверкой ОПН;
- чек-лист для КСО/КРУ дополнен проверкой видимых разрывов до и после выключателей.

## 6. Тесты

Добавлены проверки:

- расчет довеса и шин для НКУ/ЩО/ВРУ;
- расчет довеса и шин для КСО/КРУ;
- вывод количества линий, учета и ОПН в документах;
- вывод ТТ, класса ТТ, ТН, РЗА, терминала и разрывов в документах;
- сохранение новых полей панелей и ячеек в `.ktpn`.
