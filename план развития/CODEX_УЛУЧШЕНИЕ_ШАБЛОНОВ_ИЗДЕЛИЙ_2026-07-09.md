# Улучшение шаблонов изделий

Дата: 2026-07-09

Активная версия:

`versions/KtpnConfigurator_Optimized`

## Что сделано

Продолжена работа по промту:

`план развития/CODEX_ПРОМТ_УЛУЧШЕНИЕ_ПРИЛОЖЕНИЯ_2026-07-09.md`

Этот этап добавляет выбор серии и типового шаблона компоновки для новых изделий.

## 1. Версия проекта

Формат проекта обновлен:

- `ProjectConfig.CurrentVersion = 6`;
- версии данных новых изделий повышены до `2`.

Это нужно, потому что в проект добавлены новые поля серии, шаблона компоновки и расчетного КЗ.

## 2. НКУ, ЩО, ВРУ

Добавлены поля:

- серия;
- шаблон компоновки;
- расчетный ток КЗ.

Доступные шаблоны:

- `НКУ: ввод + распределение + ЩСН`;
- `НКУ: два ввода + секционный аппарат`;
- `НКУ: распределительный шкаф`;
- `ЩО: две секции 630 А`;
- `ЩО: одна секция 400 А`;
- `ЩО: две секции 1000 А`;
- `ВРУ: ввод + учет + отходящие`;
- `ВРУ: два ввода + АВР`;
- `ВРУ: учет + распределение`.

Выбор шаблона автоматически перестраивает:

- серию;
- номинал шин;
- расчетный ток КЗ;
- Icw/Ipk;
- количество секций;
- наличие АВР;
- степень защиты;
- линейку панелей.

## 3. КСО, КРУ

Добавлены поля:

- серия;
- шаблон компоновки;
- расчетный ток КЗ;
- исполнение ячеек.

Доступные шаблоны:

- `КСО: ввод + ТН + трансформатор`;
- `КСО: проходная линия + трансформатор`;
- `КСО: секционированное РУ`;
- `КРУ: ввод + ТН + секция + отходящая`;
- `КРУ: две секции с АВР`;
- `КРУ: ввод + отходящие линии`.

Выбор шаблона автоматически перестраивает:

- серию;
- номинал шин;
- расчетный ток КЗ;
- термическую и электродинамическую стойкость;
- ток отключения;
- IAC для КРУ;
- стационарное/выдвижное исполнение;
- линейку ячеек.

## 4. Интерфейс

Во вкладке `Конфигурация` добавлены:

- для НКУ/ЩО/ВРУ: строка `Серия / шаблон`;
- для НКУ/ЩО/ВРУ: поле `КЗ / Icw / Ipk`;
- для КСО/КРУ: строка `Серия / шаблон`;
- для КСО/КРУ: поле `КЗ / стойкость / время`;
- для КСО/КРУ: поле исполнения ячеек рядом с дугоотводом.

Кнопка восстановления типового состава теперь восстанавливает выбранный шаблон, а не один общий дефолт.

## 5. Проверки

Добавлены проверки:

- расчетный ток КЗ НКУ/ЩО/ВРУ не должен быть выше Icw;
- Ipk низковольтного изделия проверяется относительно расчетного КЗ;
- расчетный ток КЗ КСО/КРУ не должен быть выше тока термической стойкости;
- расчетный ток КЗ КСО/КРУ не должен быть выше тока отключения выключателей;
- Ipk КСО/КРУ проверяется относительно расчетного КЗ.

## 6. Документы и экспорт

Документы новых изделий теперь выводят:

- серию;
- шаблон компоновки;
- расчетный ток КЗ;
- Icw/Ipk;
- исполнение ячеек для КСО/КРУ.

Сводка Excel также выводит:

- серию;
- шаблон компоновки;
- КЗ/Icw/Ipk для НКУ/ЩО/ВРУ;
- КЗ/стойкость/отключение для КСО/КРУ;
- исполнение ячеек для КСО/КРУ.

## 7. Тесты

Добавлены проверки:

- шаблоны перестраивают линейки и электрические параметры;
- документы сохраняют выбранный шаблон;
- расчетный КЗ выше допустимого вызывает ошибку;
- серия и шаблон сохраняются в проекте.
