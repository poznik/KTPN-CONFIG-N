# CODEX: 2КТПН по секциям и типовые аппараты

Дата: 2026-07-09

Рабочая версия: `versions/KtpnConfigurator_Optimized`

## Что сделано

1. 2КТПН теперь считает секции Т1 и Т2 отдельно:
   - ток трансформатора по каждой секции;
   - номинал вводного аппарата секции;
   - ориентировочный ток КЗ на шинах секции;
   - шины РУНН, N и PE по каждой секции;
   - ориентировочная масса секции.

2. Валидация 2КТПН усилена:
   - ошибка, если ввод секции ниже тока своего ТМГ;
   - ошибка, если секционный аппарат ниже расчетного резервного тока;
   - предупреждение при разных мощностях Т1/Т2;
   - предупреждение при разных шинах секций;
   - предупреждение с ориентировочным КЗ секций для проверки отключающей способности вводных, секционного и отходящих аппаратов.

3. Документы обновлены:
   - приказ и паспорт показывают строки `Секция 1`, `Секция 2`, `Ориентировочный КЗ секций`;
   - спецификация добавляет `ШС1` и `ШС2`;
   - чек-лист 2КТПН дополнен проверкой отключающей способности по КЗ секций;
   - Excel-сводка показывает Т1/Т2, вводы, КЗ, шины и массу секций.

4. Вкладка результата обновлена:
   - в трансформаторах выводятся токи Т1/Т2;
   - во вводах выводится КЗ секций;
   - в шинах выводятся РУНН С1/С2, N и PE;
   - масса для 2КТПН раскладывается по секциям.

5. Для НКУ/ЩО/ВРУ/КСО/КРУ добавлены типовые списки аппаратов:
   - НКУ/ЩО/ВРУ: АВ, РПС, ПВР/NH, АВР, учет, ОПН, УКРМ, ОПС, типовые бренды и модели;
   - КСО/КРУ: ВВ/TEL-10, ВБЭ-10, ВБП-10, ВРС-10, ВНА, ВНР, РВЗ, ТТ 50/5...2000/5, классы точности, ТН, РЗА, терминалы, видимые разрывы;
   - поля сделаны редактируемыми выпадающими списками, нестандартный текст можно вписать вручную.
