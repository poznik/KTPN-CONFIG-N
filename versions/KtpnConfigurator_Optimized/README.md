# Конфигуратор КТПН Optimized

Это независимая версия приложения. Она имеет собственные исходники, тесты, сборку и не изменяет рабочую версию `build/KtpnConfigurator_V6`.

## Подготовка новых изделий

Проектная модель содержит стабильный `ProductTypeId` и отдельную версию данных изделия.
Доступны КТПН, 2КТПН, НКУ, ЩО, ВРУ, КСО и КРУ.
2КТПН использует секции и АВР, НКУ/ЩО/ВРУ - редактор панелей, КСО/КРУ - редактор ячеек.
Расчеты, проверки, документы и экспорт переключаются по выбранному изделию без изменения прежнего поведения КТПН.

## Основные отличия

- последовательный рабочий процесс с индикаторами состояния вкладок;
- схемы отключены в интерфейсе и экспорте;
- компактная таблица отходящих линий с копированием и удалением;
- автоматический подбор ввода, ТТ и зависимых номиналов;
- переход из проверки к нужной вкладке;
- расширенная расшифровка массы;
- аудит качества базы оборудования;
- единый условный пакет документов без пустых строк;
- атомарное сохранение, резервная копия и аварийное восстановление;
- обычное сохранение в текущий файл, отдельное `Сохранить как` и индикатор несохранённых изменений;
- фоновый экспорт Excel/PDF с индикатором выполнения;
- яркие статусы вкладок: заполнено, требуется проверка, есть ошибка;
- архив проекта создается рядом с экспортированными документами.
